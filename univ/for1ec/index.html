はじめに

まずは前期の授業、お疲れ様でした。慣れない大学の勉強に、これまた慣れないオンライン講義というスタイルにより、 我々2年生以上も大変ではありましたが、とりわけ1年生はかなり大変だったことと思います。 特に、授業の評価等について先輩の声を聞こうにも、そもそも繋がりができていないなど、情報収集に苦労したのではないか、と思います。 そこで僭越ながら、このような資料を作成させていただきました。同科の後輩に宛てた資料ですが、ご活用いただければ幸いです。

内容の正確性について

Writerは19ECの者で、記載の講義は全て2019年後期（対面講義）に修得した科目になります。 オンライン講義においては、その実態が大きく異なる可能性があります。また朧げの記憶を辿っている面もあるため、参考程度にご覧ください。

具体的な内容についてはなるべく正確な情報を記載するよう努めましたが、100%正確であることは保証致しかねます。 最終的な判断の前に、必ずシラバスなど、信用できるソースからの情報をご確認ください。 なお、専門用語をあえて少しだけ入れてみました。気になった科目・ワードについては、ぜひググってみてください。

他学科に配当された科目については、各学科の先輩方にお聞きください。 同じ科目名でも先生により内容が異なることもございますゆえ、一部"A先生"といった具合でイニシャルのみ表示させていただいてますが、誹謗中傷等の目的は一切ないことをここに宣言いたします。

ご不明な点などにつきまして、下記連絡先にてできる限り回答いたしますが、「本資料を参考にしたら落単した」等の内容については、ご返信いたしかねます。あくまで全て自己責任のもとでご活用ください。 上記事項に関連した嫌がらせ等については、厳に対処いたします。

人間科学科目

1EC後期においては、金2のみ受講可能です。抽選優先順位が一番高いため、履修登録することを強くお勧めします。 9/16以降は先着順のため、優先順位は関係なくなります。残定員数を眺めつつ、よく吟味しましょう。 なお人科については、他のつぶやきでまとめてあるため、省略いたします。

企業と経営 / 金2

英語科目

前半は受講していましたが、TOEICによる単位認定で修得してしまったため、後半の内容はわかりません。前期と同じ先生のはずであり、内容もその延長ですので、割愛させていただきます。 （内容については、履修クラスによる差も大きいようですね。） 後期は学内TOEICを実施するようですので、自信のある方は単位認定の利用をお勧めします。 ちなみに対面講義においては、少人数クラスでグループワークなどのアクティブ・ラーニングが活発に行われるため、コミュニケーションの輪を広げることができました。(数・英はEC/FI科が合同でした)

総合英語II / 火3

口語英語II / 金4

… Thanks God, It’s Friday.

工学基礎必修科目

前期でほとんど終わりなので、再履修等を除けば1ECの場合、実験科目のみが残っているはずです。 実験においては、計算式に気温・気圧など気象条件がパラメータとして登場したりと、かなり厳密に求める場面が多いです。 （レポートに定量的な考察が求められることとも関係していると思います。）料理ではないので、ことわりのない限り目分量はやめましょう。

レポートが重かったりしますが、誰もが逃げられないため、諦めて受け入れましょう。私も毎週末〜実験まではレポートについて毎回呟いていた気がします。 （物理実験は2週間・化学実験は1週間がレポートの締め切りであり、提出期限日が重なります。 そのため、計画を立てて進めないと、週末になってこう嘆くことになりますーー*「何てことだ、もう助からないゾ」*ーー。）

物理実験 / 月1, 2（隔週）

測定器具操作＋実験4回を行いました。実験題目によりますが、実験自体は面白いものが多いです。 （高校物理で習ったことや、前期「基礎物理学」で習ったことをはじめ、種々の法則について、実験により確認することが主な内容です。）

2019年の実験書には、実験題目が9題目記載されていました。そのうちの4題目（選択不可）について、2-3人の少人数の班で実験を行い、レポートを作成するのが主な流れです。 初見のギリシャ文字が登場したりして、一見複雑そうな数式が散見されますが、実験中の計算自体は関数電卓に任せられるので心配ご無用です。 ただし、レポートにおいては、導式等でつらつら書かなければならないこともあるので、数式アレルギーの方はちょっと大変かもしれません。

何より、手書きレポートという点が一番辛い気もします。図を書く機会はそこまで多くないので、その点はワクショに比べたら楽かもしれませんが。 修正の効かないグラフ等については特に気を付けましょう。もし清書でミスしてしまうと…0% 0% 0%…。

参考文献として、Webページ不可だったため、実験書や高校までの物理の教科書・参考書を使いまくった気がします。

化学・生物実験 / 月1, 2（隔週）

化学実験

実験は全3回＋PCを利用した分子の可視化が1回。 金属の分離・検出のプロセスやダニエル電池など、高校化学までで見かけたような内容が登場します。 PCを利用した分子の可視化については、英語でのソフトウェア利用が強いられます。辞書の利用は可能なため、専門用語については都度調べれば問題はあまりありませんが、 文法もままならないとかなり苦しいと思います。配布資料も英語のため、TOEIC対策がてら、頭の隅に置いておくと良いと思います。

ワープロでのレポートがあります。検印欄に再再提出まであり、些細なミスでもチェックされるため、再々レポートを課されてる学生は多かった気がします。 ワープロ作業ならではのミス（5.00 mLといった数値と単位間の空白抜けなど）まで事細かに見られるため、よーく睨めっこしましょう。（それでも見落とすのですが…）

レポートはテンプレートがあるため、目的・理論・方法については穴埋めで記述していくことができますが、結果・考察については一から文章を組み立てていく必要があります。

生物実験

オアシス。おそらく実験の中で最も気楽に受けられるかと思います。予復習の確認テストがありますが、基本的にレポートが課されません。 例外的に、「遺伝子のDNA」については考察の記述を求められましたが、他の実験科目ほど重いものではありません。（早い人であれば講義時間内に書き終えられるはずです）

実験は全2回4題目で、1回につき2題目実施します。

工学基礎選択科目

自然科学概論です。前期に1科目以上取った方は無理に履修する必要はないです。（「卒業要件：工学基礎科目20単位以上」は自然科学概論1科目+必修で満たすことができます。）

ちなみに、1→2年の進級条件は、 「1年次終了時に、自由科目を除く修得単位数の合計が24単位以上であること」です。 (ref. 学生要覧 工学部 2020 p.117)

UNIPAで履修登録時に進級見込み判定が行えますので、ご確認ください。また現時点での進級見込判定・卒業見込判定も時間割＞学生時間割から行うことができます。 卒業要件以外に自然科学概論が足枷となることはありませんが、年次ごとに抽選優先順位がつけられており、「（高）1,2 > 3 > 4（低）」であるため、早めに履修することを推奨します。

自然科学概論C / 月1

2019年度前期に履修したため、ここでは省略します。前期版の資料をご覧ください。

自然科学概論G / 金1

（同上）

専門必修科目

後期より、いよいよEC科固有の科目（専門科目）の講義も始まります。と言っても、必修科目は必修科目なので、時間割に融通を効かせることはできませんが。 「このコマは必修用だ、食べられないよ」

ちなみに、Twitterでは#必修1限を許すなというハッシュタグがあります。1EC後期においては3科目該当しますが、オンラインにおいては絶起リスクが低いことは幸いですね。

情報通信メディア基礎 / 火1

自然科学概論GのEC科特化バージョンです。既に修得された方はこれで伝わると思います。 各回異なる先生が、研究分野についてその「さわり」の部分を紹介する講義です。研究室紹介に近いかもしれません。 様々な分野についての話が聞けるため、興味を広げられる他、将来卒業研究を行う際の参考になるかと思います。

電気回路基礎及び演習 / 水1, 2

高校物理の「電磁気」の延長です。フェーザ表示など、より工学的なものを学んでいきます。 演習(1限)は隔週に近い頻度のため、セメスタの半分程度は2限のみで終わります。

コンピュータプログラミングII / 月3, 4

C言語をさらに深く学びます。前期「コンプロI」の内容の復習も含みます。 おそらく、「ポインタ」が鬼門になると思います。しっかり予復習しておきましょう。 対面講義時は、話を聞いた後に、問題を解いて提出した時点で終了でしたので、早く切り上げられることも多かったです。

ワークショップII / 木3, 4

内容は前期の延長。レポートも変わらず手書きでしたが、記述量が多くなります（他の実験科目同様、体裁をしっかりする必要がある）。 レポートは提出したその日のうちに返却されるため、空き時間ができ、図書館で時間を潰していました。2020年後期はこの形態とも限りませんね。

専門選択科目

一番自由に組み替えることができ、「ぼくの考えた最強の時間割」を作り出すための腕の見せ所です。 特にEC科は広範な分野に渡りますので、選択する科目は将来も見据えて慎重に吟味しましょう。尤も、1EC後期に配当される科目は多くの分野につながるため、すべて履修することをお勧めしますが。

高学年で楽をしたい・短期卒業を目指したいならば、各学期20単位以上履修登録することをお勧めします。 （履修上限+4単位の恩恵を受ける条件が、前学期20単位以上履修登録かつ90%以上の単位修得に加え、GPA3.1以上であるからです）

逆に、修得している単位数があまりに少なすぎる場合、後々辛くなるばかりか、奨学金の打ち切り等様々な悪影響を及ぼすため、ご注意ください。 私も「夏休みの宿題は最終日間近にまとめてやる派」でしたが、履修計画に関してはかなり詰め込んでいる方だと自負しています。 今期はあまり詰め込めそうにありませんが...。

微分積分学および演習II / 火4, 金3 / I先生

科目名のごとく、前期「微分積分学I」の延長です。かなり丁寧に解説してもらえるため、非常に理解しやすかった思い出があります。 新たに学ぶのは、「偏微分」「全微分」「多重積分」「広義積分」です。関連して、「ヤコビアン」として線形代数学の内容が一部含まれます。

線形代数学II / 火2 / K先生

計算量がとても多くなります（「余因子展開」「固有値と固有ベクトル」など）。つらいさんなのだ。 ∀, ∃と言った記号が当たり前のように登場するため、慣れないうちは大変かもしれません。 また、長い証明もあります。試験には出されませんでしたが。 実際に計算を行うことで「なるほど、そういうことか」となりやすい科目ですので、他の科目以上に復習をしっかりすることをお勧めします。（私と同じ轍を踏まないよう…）

情報通信数学A（離散数学）/ 金1

性質上、記号がたくさん出てきます。数式・命題そのものは簡潔だったりするものもありますが、その内容を解釈するのに時間がかかるものもあります（逆も然り）。 高校までに扱った「数学」とは、少し毛色が違うかもしれません。一応扱っている部分もありますが…。 「グラフ理論」「組み合わせ理論」「最適化問題」などが絡む分野であるという認識が強く、競技プログラミングに取り組んでいる身としてはお勧めしたい科目ではあります。 （ただし本科目内でグラフ理論については扱わなかったことを補足しておきます。今までの数学科目同様、一つの分野について体系的に学んでいくことに変わりありません。）

完全な余談ですが、競技プログラミング(競プロ)についても勧めておきます。アルゴリズムについてパズルチックに鍛えられて良いですし、コンプロの予復習にもなります。
